# Análisis de Movilidad Urbana y Productividad Económica
### Banco Interamericano de Desarrollo (BID) | TripleTen Bootcamp 2026

## 📌 Contexto
Como analista de datos para el BID, el objetivo fue entender cómo la movilidad urbana (congestión, tiempos de viaje) se relaciona con la productividad económica (PIB per cápita, desempleo) en las principales ciudades del mundo, para identificar dónde invertir en infraestructura de transporte.

## 🎯 Preguntas de negocio
- ¿Qué ciudades presentan alta congestión y baja productividad económica?
- ¿Cuáles muestran los mejores indicadores combinados?
- ¿Qué variables tienen mayor relación con el desarrollo urbano?

## 🔍 Hallazgo principal
Colombia presentó los peores índices de congestión en correlación con bajo PIB, 
lo que derivó en una recomendación accionable de priorización de inversión en 
infraestructura vial.
